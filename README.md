#What is MERN stack?

**MERN** stands for **MongoDB**,**Express**,**React**,**Node** where.

1.  MongoDb - document based Database
2.  Express(js) - node.js web framework
3.  Reactjs - a client side javascript framework
4.  Nodejs - JavaScript web server
